# Visual assets, AI и новые продукты

[← Оглавление](README.md)

Figma давно не ограничивается static UI screens. Внутри ecosystem есть vector
tools, image workflows, export, Figma Draw, Motion, AI, Make, Sites, Buzz and
Slides. Эта глава собирает визуальные и новые "крутые" фичи.

## Fills, strokes and effects

`daily`

Visual design in Figma often starts with layer appearance.

Fills:

- solid color;
- image/video fill;
- gradients;
- multiple fills;
- opacity;
- blend modes.

Strokes:

- color;
- weight;
- position;
- dash;
- end caps;
- joins;
- advanced stroke settings.

Effects:

- drop shadow;
- inner shadow;
- layer blur;
- background blur;
- effect styles.

## Vector networks

`advanced`

Figma vector networks are flexible paths that can branch, unlike many traditional
single-direction vector tools.

Use cases:

- icons;
- custom shapes;
- illustrations;
- diagrams;
- hand-drawn details.

Related tools:

- Pen;
- Pencil;
- Brush;
- vector edit mode;
- bend;
- cut;
- lasso;
- paint;
- variable width;
- shape builder.

## Boolean operations

`daily/advanced`

Boolean operations combine shapes.

Operations:

- union;
- subtract;
- intersect;
- exclude.

Use cases:

- icons;
- masks-like shapes;
- badges;
- cutouts;

## Masks

`daily`

Masks control what part of another layer is visible.

Use cases:

- avatar crop;
- image card;
- circular thumbnail;
- clipped illustration;
- reveal effects.

## Images and media

`daily`

frames and text layers that support fills.

Useful settings:

- crop;
- fit/fill behavior;
- replace image;
- image opacity;
- blend modes;
- masks;
- export.

Use cases:

- product screenshots;
- avatars;
- maps;
- hero images;
- thumbnails;
- background textures;
- prototype realism.

## Figma Draw

`new/cool`

Figma Draw adds expressive illustration tools directly inside Figma Design.

Capabilities:

- brushes;
- transforms;
- textures;
- vector editing improvements;
- custom brush styles;
- pattern-style visual work;
- illustration-focused workflow.

Why cool: it brings more expressive, handmade visual work into the same canvas
where product UI already lives.

Examples:

## Figma Make

`new/cool`

Figma Make is an AI-driven prompt-to-app tool for functional prototypes, web
apps and interactive UI.

Capabilities:

- start from prompt;
- start from existing Figma design;
- chat with AI to iterate;
- preview functional prototype/web app;
- point and edit;
- edit code directly;
- attach Figma designs;
- bring style context from libraries where available;
- add backend in supported workflows;
- publish/share.

Why cool: it connects design exploration to functional behavior much faster than
static prototyping alone.

`new/cool`

Capabilities:

## Visual feature checklist

- Use vector networks for reusable icons and detailed shapes.
- Use boolean operations for icon construction.
- Use masks for image cropping.
- Use images as fills, not as a separate layer model.
- Add export settings to production assets.
- Use Dev Mode for asset handoff.
- Use Figma Draw when the design needs expressive illustration.
- Use Motion/prototype animations for behavior, not just decoration.
- Use AI/Make/Sites/Buzz when the workflow benefits from generation,
  publishing or scaled asset production.
