# Цель

Довести подсистему аннотаций дизайн-документа до готового PR в ветке
`claude/awesome-cartwright-05e2a7`.

Функциональный результат:
- чистое ядро `:subsystems:annotations` с моделью, операциями, геометрией маркера и экспортом issue в промпт;
- Compose-рендерер `:subsystems:annotations-compose` с каплей, раскрытой табличкой, вложенной картинкой и overlay поверх артборда;
- sidecar-формат `:subsystems:annotations-slm` для `*.annotations.md` с parse/write/patch;
- интеграция в `:shared`: состояние документа и вида, интенты редьюсера, write-back в sidecar, overlay в canvas, inspector UI и экспорт замечаний;
- тесты и сборка зелёные, ветка синхронизирована с актуальной целевой веткой, создан готовый PR.

Ограничения v1:
- sidecar остаётся отдельным review-слоем рядом с SLM и не встраивается в `*.layout.md`;
- issue подсвечивается через `statusWarning` и попадает в промпт, note не экспортируется;
- freehand-ink, RichText тела и кросс-экранные якоря вне текущего объёма.
